# Informative-Censoring Variants — Coverage Study

Conditional non-informative (covariate-dependent) censoring stress test for
the recurrent-event ODE estimator. For every subject, the recurrent event
process and the censoring time share the same covariates `x` but are
generated by independent random draws given `x`. Marginal correlation
between `C` and `N(.)` arises only through `x`.

The estimator code is unchanged across all six variants — only the
censoring distribution differs.

## Setup

- N = 1000 subjects, 100 replications each.
- Covariates: `x1, x2 ~ N(0,1)` clipped to `[-1, 1]`, `x3 ~ Bernoulli(0.5)`.
- Truth: `beta = (1, 1, 1)`. Inference reports Wald 95% CIs from sandwich SE.
- Cox event intensity (setting 1):
  `lambda(t|x) = (t^2 + 1) * exp(x'beta)`
- AFT event intensity (setting 2):
  `lambda(t|x) = 2 * exp(x'beta) / sqrt(4 * exp(x'beta) * t + 1)`
- gamma = (1, 1, 1) for the censoring linear predictor in all variants.
- Final follow-up `u_i = min(C_i, tau)` for survival-type variants
  (`tau = 4`), and `u_i = C_i` for the uniform variant. A 1e-6 jitter is
  added only to subjects sitting at the cap / numerical floor so that the
  downstream `solve_ivp` sees a strictly-increasing `t_eval`.

## Three censoring families

### 1. Cox-survival censoring

```
h_C(t|x) = lambda_0 * exp(x' gamma),   lambda_0 = 0.3
```

`C|x ~ Exp(0.3 * exp(x' gamma))`. Inversion: `C = -log(U) / (0.3 * exp(x'gamma))`,
`U ~ Uniform(0,1)`. Constant-hazard exponential.

### 2. AFT-survival censoring

```
h_C(t|x) = 2 * lambda_0 * exp(x' gamma)
           / sqrt(4 * lambda_0 * exp(x' gamma) * t + 1),   lambda_0 = 0.2
H_C(t|x) = sqrt(4 * lambda_0 * exp(x' gamma) * t + 1) - 1
```

Inversion: `V ~ Exp(1)`, `C = (V^2 + 2V) / (4 * lambda_0 * exp(x' gamma))`.
Decreasing hazard.

### 3. Uniform censoring

```
C | x ~ Uniform(a(x), b(x))
a(x) = 0.2     (constant lower bound)
b(x) = 4 - 0.5 * (x' gamma)
```

With `x' gamma` in `[-2, 3]`, `b(x)` in `[2.5, 5]`, always > `a(x)`.
Generated by `C = a(x) + (b(x) - a(x)) * U`, `U ~ Uniform(0,1)`. Bounded
support — qualitatively different from the two survival models.

In every variant, `C` is generated from `x` and an independent uniform draw,
so `C` and the recurrent-event process are independent given `x` — the
estimator's required assumption.

| Setting | mean_est | bias | emp_se | mean_se | coverage95 |
|---|---|---|---|---|---|

| `inform_aft`      (AFT events + Cox-surv C)    | [0.997, 0.994, 0.982] | [-.003, -.006, -.018] | [.044, .043, .064] | [.046, .046, .065] | [0.95, 0.96, 0.96] |
| `inform_cox_aftc` (Cox events + AFT-surv C)    | [0.999, 1.000, 1.000] | [-.001,  .000,  .000] | [.015, .018, .021] | [.015, .015, .020] | [0.94, 0.92, 0.95] |
| `inform_aft_aftc` (AFT events + AFT-surv C)    | [0.994, 0.993, 0.984] | [-.006, -.007, -.016] | [.045, .044, .065] | [.048, .048, .067] | [0.95, 0.95, 0.97] |
| `inform_cox_unif` (Cox events + Uniform C)     | [1.001, 1.000, 0.999] | [ .001,  .000, -.001] | [.013, .013, .018] | [.014, .013, .018] | [0.94, 0.95, 0.93] |
| `inform_aft_unif` (AFT events + Uniform C)     | [0.985, 0.991, 0.984] | [-.016, -.009, -.017] | [.038, .041, .061] | [.045, .045, .062] | [0.97, 0.96, 0.95] |

Coverage stays near the nominal 0.95 across all three censoring families.
The censoring shape is irrelevant — what matters is conditional
independence, which all three variants satisfy.

## Censoring profile (single seed, N = 1000)

| Setting | row%cens | mean_evt | %0-evt subj | %u≈4 | mean follow-up |
|---|---|---|---|---|---|
| `inform_cox`       | 10.5% |  8.55 | 18.1% | 24.6% | 1.80 |
| `inform_aft`       | 31.7% |  2.16 | 22.7% | 24.6% | 1.80 |
| `inform_cox_aftc`  |  8.0% | 11.58 | 20.5% | 29.0% | 1.88 |
| `inform_aft_aftc`  | 30.8% |  2.24 | 23.7% | 29.0% | 1.88 |
| `inform_cox_unif`  |  6.3% | 14.87 | 10.4% |  3.8% | 1.97 |
| `inform_aft_unif`  | 23.8% |  3.21 | 15.6% |  3.8% | 1.97 |

- **row%cens**: fraction of long-format rows with `delta=0` (each subject
  contributes 1 censoring row + `n_events` event rows). Lower for Cox-event
  settings because each subject accumulates many event rows.
- **mean_evt**: mean number of recurrent events per subject.
- **%0-evt subj**: fraction of subjects with no observed events at all
  (the closest analog to "100% censored" in single-event survival).
- **%u≈4**: fraction of subjects whose follow-up is at the admin τ=4.
  For the survival-type variants this is the admin-cap rate; for Uniform
  it is just `P(C >= 4) ≈ 3.8%` since `b(x) <= 5`.
- **mean follow-up**: average `u_i`. The Uniform variant has slightly
  longer follow-up because its support `[0.2, 5]` is more uniform than
  the right-skewed survival distributions.

The cross-event-shape comparison (Cox vs AFT recurrent-event intensity)
explains the row%cens spread: Cox intensity grows over time and produces
many events per subject; AFT intensity decays, giving few events.

## Code layout

```
inform_censor/
  cox/        Cox events       + Cox-survival censoring  (lambda_0 = 0.3)
  aft/        AFT events       + Cox-survival censoring  (lambda_0 = 0.3)
  cox_aftc/   Cox events       + AFT-survival censoring  (lambda_0 = 0.2)
  aft_aftc/   AFT events       + AFT-survival censoring  (lambda_0 = 0.2)
  cox_unif/   Cox events       + Uniform censoring       (a=0.2, b=4-0.5*x'g)
  aft_unif/   AFT events       + Uniform censoring       (a=0.2, b=4-0.5*x'g)
```

Each variant has `generator_rec.py` (data) and `main.py` (driver). The
fit/inference code is reused unchanged from `RecurrentODE_py.cox.*` /
`RecurrentODE_py.aft.*`.

## Reproducing

```bash
# 100-rep coverage for any variant:
python3 -m RecurrentODE_py.test.test_coverage \
    --module inform_censor.cox      --setting 1                   --N 1000 --rep 100 --workers 10
python3 -m RecurrentODE_py.test.test_coverage \
    --module inform_censor.aft      --setting 2 --knots equal     --N 1000 --rep 100 --workers 10
python3 -m RecurrentODE_py.test.test_coverage \
    --module inform_censor.cox_aftc --setting 1                   --N 1000 --rep 100 --workers 10
python3 -m RecurrentODE_py.test.test_coverage \
    --module inform_censor.aft_aftc --setting 2 --knots equal     --N 1000 --rep 100 --workers 10
python3 -m RecurrentODE_py.test.test_coverage \
    --module inform_censor.cox_unif --setting 1                   --N 1000 --rep 100 --workers 10
python3 -m RecurrentODE_py.test.test_coverage \
    --module inform_censor.aft_unif --setting 2 --knots equal     --N 1000 --rep 100 --workers 10
```

Per-seed `seed*.npz` and aggregated `_summary.npz` for every variant are
under `RecurrentODE_py/test/results/inform_censor_*/`.

## Caveat

The current theoretical guarantees do not formally cover covariate-
dependent censoring. The empirical coverage above (all near 0.95) is
evidence that the estimator behaves robustly under this departure, at
least when the conditional-independence-given-`x` assumption still holds
and the marginal C/N(.) coupling is moderate.
